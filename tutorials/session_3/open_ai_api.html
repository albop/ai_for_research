REST API

The openAI services are accessible through a REST API. REST is a protocol designed to call services (through) http requests (POST and GET requests).

# example of a an api call (Chat API - current standard)
import requests
key = api_key
url = "https://api.openai.com/v1/chat/completions"
headers = {"Authorization": f"Bearer {key}", "Content-Type": "application/json"}
data = {
    'model': 'gpt-3.5-turbo', 
    'messages': [{'role': 'user', 'content': 'Once upon a time'}]
}
requests.post(url, headers=headers, json=data).json()

Preference Survey Module

The The Preference Survey Module: A Validated Instrument for Measuring Risk, Time, and Social Preferences by Falk et al has created a qualtative survey meant to measure preferences of humans in a such a way that it replicates incentivized experiments carried on in laboratories.

In AI responses to the preference survey module Mimra and Winant, have adapted the same survey to measure the preference of common large language models.

The goal here is to replicate one of the experiments to measure altruism.

Two questions are asked:

  • Imagine the following situation: you won 1,000 dollars in a lottery.Considering your current situation, how much would you donate to charity?

  • How do you assess your willingness to share with others without expecting anything in return when it comes to charity? Respond on a scale from 0 0 to 10, where 0 means you are ‚Äúcompletely unwilling to share‚Äù and a 10 means you are ‚Äúvery willing to share‚Äù. You can also use the values in-between to indicate where you fall on the scale.

Eeach response should be normalized between 0 and 10. Then multliply the first response by 0.1845 and add the second response multiplied by 0.3210, to get the final evaluation.
